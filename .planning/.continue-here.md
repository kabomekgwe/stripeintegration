- **Cache 5 minutes**: Balance between freshness and API limits (100 ops/sec live)

- **Show base_rate**: Mid-market rate, not the rate with Stripe's ~2% FX fee

<context>
**FX Quote Structure (from Stripe):**
```json
{
  "id": "fxquote_xxx",
  "object": "fx_quote",
  "to_currency": "usd",
  "rates": {
    "gbp": {
      "exchange_rate": 1.10167,
      "rate_details": {
        "base_rate": 1.12415,      // Mid-market (USE THIS)
        "fx_fee_rate": 0.02,        // Stripe's 2% fee
        "reference_rate": 1.12437   // ECB/market reference
      }
    }
  }
}
```

**Regulatory Disclaimers Required:**
- EU (Regulation 2019/518): Show exchange rate source, freshness, and estimate disclaimer
- UK (FCA Consumer Duty): Clear that rate is estimate, final at transaction time
- Default: "This is an estimate. Actual charge may vary."

**Currency Decimal Precision (ISO 4217):**
```typescript
const CURRENCY_DECIMALS = {
  // Zero-decimal
  JPY: 0, KRW: 0, CLP: 0, VND: 0, ISK: 0,
  // Three-decimal
  BHD: 3, KWD: 3, OMR: 3, JOD: 3, TND: 3, LYD: 3, IQD: 3,
  // Default: 2
};
```

**Current Flow:**
1. User enters amount in GBP
2. Frontend detects user's currency from country/preference
3. Frontend calls `/currency/convert?amount=X&from=gbp&to=usd`
4. Backend returns converted amount + rate
5. Frontend shows "≈ $127.50 USD" (basic)

**Enhanced Flow:**
1-4: Same
5. Frontend shows full conversion card with rate source, freshness, disclaimer
