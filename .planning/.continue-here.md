<decisions_made>
- **Use Stripe FX Quotes API**: Business requires currency conversion preview in Step 1. Must use ONLY Stripe (no external FX APIs). FX Quotes is a preview API that provides mid-market exchange rates.
- **Use `base_rate` not `exchange_rate`**: `base_rate` is mid-market rate (no fee), `exchange_rate` includes Stripe's ~2% FX fee. We use `base_rate` for display.
- **Type assertion for preview API**: TypeScript SDK lacks `fxQuotes` types, so we cast `this.stripe` to access it.
- **Inverse the rate**: FX Quotes returns "how much USD for 1 foreign currency", but we need "how much foreign currency for 1 USD", so we use `1 / base_rate`.
</decisions_made>


**Rate structure:**
```json
{
  "rates": {
    "gbp": {
      "exchange_rate": 1.10167,  // includes 2% fee
      "rate_details": {
        "base_rate": 1.12415,      // mid-market (use this)
        "fx_fee_rate": 0.02,
        "reference_rate": 1.12437
      }
    }
  }
}
```
