    2. In main.ts, add Swagger configuration BEFORE app.listen():
       ```typescript

       const config = new DocumentBuilder()

         .setDescription('API documentation for Stripe Platform')
         .setVersion('1.0')
         .addBearerAuth()
         .build();

       const document = SwaggerModule.createDocument(app, config);
       ```

<task type="auto">
  <name>Task 3: Add @ApiOperation and @ApiResponse to endpoints</name>
  <files>backend/src/auth/auth.controller.ts, backend/src/payments/payments.controller.ts, backend/src/subscriptions/subscription.controller.ts</files>
  <action>
    For the three most critical controllers (auth, payments, subscriptions), add detailed API documentation:

    1. Import decorators:
       ```typescript
       import { ApiOperation, ApiResponse, ApiBody, ApiParam, ApiQuery } from '@nestjs/swagger';
       ```

    2. For each endpoint method, add:
       - @ApiOperation({ summary: 'Description' })
       - @ApiResponse({ status: 200, description: 'Success', type: ResponseDto })
       - @ApiResponse({ status: 400, description: 'Bad request' })
       - @ApiResponse({ status: 401, description: 'Unauthorized' })
       - @ApiBody for POST/PUT endpoints
       - @ApiParam for path parameters
       - @ApiQuery for query parameters

    3. Example for auth controller login:
       ```typescript
       @Post('login')
       @ApiOperation({ summary: 'Login user and return JWT tokens' })
       @ApiBody({ type: LoginDto })
       @ApiResponse({ status: 200, description: 'Returns access and refresh tokens', type: AuthResponseDto })
       @ApiResponse({ status: 401, description: 'Invalid credentials' })
       async login(@Body() dto: LoginDto) { ... }
       ```

    4. Document at least:
       - Auth: login, register, refresh, logout, forgot-password, reset-password
       - Payments: create-payment-intent, confirm, list, get
       - Subscriptions: create, list, get, cancel, update
  </action>
